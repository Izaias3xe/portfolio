## ✨ Sobre o Projeto

Meu portfólio foi criado com o objetivo de transmitir profissionalismo, criatividade e presença visual forte, destacando:

- Minha apresentação pessoal
- Tecnologias que domino
- Projetos desenvolvidos
- Experiência com desenvolvimento Front-End
- Design moderno e minimalista
- Interface responsiva e elegante
- Animações suaves e experiência visual refinada

A ideia principal foi criar algo que representasse minha identidade como desenvolvedor e causasse uma ótima primeira impressão.
